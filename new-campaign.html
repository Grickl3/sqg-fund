Start a Campaign

Be a part of the experiment to surpass the dreams
of both Capitalism and Socialism

February 12, 2019 — Deacon Rodda

To be really, super, absolutely upfront, as of today, you are almost certainly better off using one of the big four crowdfunding platforms. They have integrated Content Management Systems, real-time display for campaign contributions and contributor activity, and a hundred other features which are not yet built out for this system. It is also worth pointing out that Slow Spread Sink is not being built because those crowdfunding systems are bad. They aren't. Each of them solves a problem and lives in a much needed niche. They are more culturally accessible to the general public than SSS aims to be, and they have superb development teams.

If you do want to start a Slow Spread Sink campaign …

You are certainly allowed. The whole project has been public since the first line of code was written. All you'll need is a basic knowledge of web development or a geeky friend. The repository can be found here. But, for real, it isn't all that great yet. Hell, if you are so inclined, maybe you'll help us build it.

Slow Spread Sink is being built because there is farther to go. Because there are flaws and draw-backs in every system (this one will be no exception). And, SQGLZ aims to fill some of the meaningful gaps in venture provisioning, especially for those who are ready to move past the destructive twentieth century paradigms of Capitalism and Socialism.
